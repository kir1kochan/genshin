# 开发任务清单

## 任务模块划分

### 1. 任务系统
- [ ] 完善主线任务触发和完成逻辑与流程。
- [ ] 设计主线剧情的对话系统和相应的对话文本。
- [ ] 确保任务与地图动态交互（如地图点位解锁、触发事件）。
- [ ] 添加多样化支线任务模板（探索、护送、收集）。
- [ ] 设计至少 3 钟支线任务的运作方式。

### 2. 地图完善
- [ ] 增加对象层的碰撞区域类型字段。
- [ ] 在对象层增加采摘烹饪和钓鱼点的点位标记与相应需要的特殊字段，对应的贴图直接加在地图上。
- [ ] 在对象层增加刷怪的点位标记与相应需要的特殊字段，不需要怪物贴图。
- [ ] 地图与主线对接。

### 3. UI 和美术
- [ ] 角色面板和背包界面设计，以及任务界面设计并完善对应的交互功能。
- [ ] 添加动态地图 UI（如迷雾探索、传送点和小地图）以及对应的联动效果。
- [ ] 搜集角色和怪物，技能和装备图标以及需要的动画，放在相应的文件夹中。
- [ ] 设计游戏操作手册。

### 4. 测试与优化
- [ ] 加入战斗系统，地图探索与交互系统，任务系统的测试。
- [ ] 优化可能存在的卡顿现象，整体测试时可以考虑加入debug模式用于测试潜在的问题。

### 5. 可选功能
- [ ] 加入音效和音乐，并添加相应的设置界面用于控制音量。
- [ ] 加入天气系统并完善对应的实现。
